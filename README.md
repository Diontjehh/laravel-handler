# Laravel Handler

## Installation

You can install the package via composer:

```bash
composer require diontjehh/laravel-handler
```

## Usage

```php
CreateUserAction::run([
    'name' => 'John Doe',
    'email' => 'john@example.com',
]);
```

## Testing

```bash
composer test
```
